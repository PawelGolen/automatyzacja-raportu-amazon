# Amazon Sales Report Automation (Excel VBA)

An open-source project that automates the processing of **Amazon Seller Central sales reports** using an Excel workbook with macros (`.xlsm`).  
The file enables fast data cleaning, analysis, and generation of clear, business-ready summary reports.
